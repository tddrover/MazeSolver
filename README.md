# Maze Solver 

A c++ program that uses maze information from a file, finds the entrance, solves the maze using a stack-based algorithm, and prints the solution path. It utilizes custom stack data structures and node representations for maze traversal.

Includes:
- Text file of 1s and 0s that make up the "maze"
- File that represents a stack data structure
- File that declares StackNode which represents a node in a stack
- Main program that solves the maze
